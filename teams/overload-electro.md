# Overload (Electro)

**Overload** occurs when a **Pyro** marked enemy is hit with **Electro** or vice versa, creating a small **AoE** hit of **Pyro damage** and can potentially knock back enemies. **Overload damage** is based off the **Elemental** **Mastery** and level of the character applying the second element. Most **Overload** teams have a **Pyro** Main DPS, however several on-field **Electro** Main DPS characters can work with Xiangling effectively.

You can think of team building for overload like this:

|                                   |                               |                                |
| :-------------------------------: | :---------------------------: | :----------------------------: |
|                                   | **On-Field Pyro Application** | **Off-Field Pyro Application** |
|  **On-Field Electro Application** |         Does not work.        |              Works             |
| **Off-Field Electro Application** |             Works             |          Does not work         |

Carries represent on-field application of their respective elements (electro for Keqing, pyro for the others). After those two roles are fulfilled, the remaining slots are there to add additional damage/supplement survivability.

## Sample Team Compositions

![](../.gitbook/assets/UI\_AvatarIcon\_Keqing.png)![](../.gitbook/assets/UI\_AvatarIcon\_Xiangling.png)![](../.gitbook/assets/UI\_AvatarIcon\_Venti.png)![](../.gitbook/assets/UI\_AvatarIcon\_Bennett.png)

* The goal is to alternate between using Xiangling burst and a **Pyro** infused Venti **Burst** to constantly apply **Pyro** to the enemy, then using Keqing’s infused **Normal** and **Charged attacks** to apply **Electro** while Xiangling’s Pyronado is active, and using her **Burst** and **Skill** while Venti’s infused **Burst** is active.
